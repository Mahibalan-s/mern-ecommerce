# ProShop MERN Stack E-commerce

A full-featured shopping cart application built with the MERN stack (MongoDB, Express, React, Node.js).

##  Features
- **Full-stack MERN:** React-Bootstrap frontend, Node/Express backend.
- **State Management:** Redux Toolkit for cart, user auth, and orders.
- **Authentication:** Secure JWT-based login with HTTP-only cookies.
- **Payments:** Integrated with **PayPal** and **Razorpay** SDKs.
- **Admin Tools:** Dashboard to manage products, users, and order status.

##  Tech Stack
- **Frontend:** React, Redux, React-Bootstrap, Vite
- **Backend:** Node.js, Express, Mongoose
- **Database:** MongoDB
- **Security:** JSON Web Tokens (JWT), Bcrypt.js
